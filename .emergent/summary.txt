The user wants to create a web-based application called AdVantage Studio to visualize ad previews across various social media platforms. The application should help users create high-converting and engaging ads by allowing them to customize every aspect and see a live preview.

- **Core Functionality:** A dashboard to create and preview social media ads.
- **Inputs:** Users can add visuals (images, videos), captions, headlines, and CTAs.

- **Collaboration:** A feature to generate a shareable link for an ad preview.
